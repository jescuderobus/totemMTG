Volver

Screensaver

Notes:

The idea for this pen originates from a pen by Scott Kelum.

Important: Make sure that the background colour is set on the inner element, not the parent!

This effect is based on animating the x and y positions of an element and its inner element. If you want to have something animating in some way, for example, a rotating image, or scaling, code in an additional element within the inner element and perform the animation on that.

It's possible to play around with the duration of the animation as well as things like easing.

There is no real application for this, apart from some kind of screen saver, or perhaps a loading screen. I have used this in a 404 page with a book and it's pages floating around. The primary use is just for fun!
